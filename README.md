Detect whether a news article is FAKE or REAL using a machine learning model.

## Dataset:
- Kaggle Fake News Dataset: https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset

## Tools:
- Python
- Scikit-learn
- Pandas
